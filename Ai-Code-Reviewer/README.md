# AI Code Review Application

A locally-run AI-powered code review application using DeepSeek Coder 6.7B model.

## Features

- **Dashboard**: Overview of code review statistics
- **AI Code Review**: Submit code for AI analysis and review
- **Style Guides**: Manage coding standards and guidelines
- **Training Data**: Import past PRs for AI training
- **Settings**: Configure application preferences

## Prerequisites

1. Python 3.8+
2. Ollama (for running DeepSeek model locally)
