Unsupervised Learning

Discover hidden patterns and structures in data

What is Unsupervised Learning?

It's a machine learning approach where models find patterns in data without labeled outputs. The algorithm explores the data structure to discover hidden relationships, groupings, or reduce dimensions. This is like learning by observation without a teacher, finding natural patterns on your own.

Available Algorithms
  • K-Means Clustering
  • PCA
  • DBSCAN
K-Means Clustering

Group similar data points together by iteratively assigning points to nearest centroids.

  • Centroid-based clustering
  • Requires number of clusters (k)
  • Fast and scalable
Explore K-Means
Principal Component Analysis

Reduce data dimensions while preserving maximum variance. Transform 3D data to 2D or 1D.

  • Dimensionality reduction
  • Feature extraction
  • Visualization aid
Explore PCA
DBSCAN Clustering

Density-based clustering that finds arbitrary-shaped clusters and identifies noise points.

  • Arbitrary cluster shapes
  • Noise detection
  • No need for cluster count
Explore DBSCAN
