Algorithm Information

DBSCAN Steps:

  1. Find core points (minPts in ε-radius)
  2. Expand clusters from core points
  3. Connect density-reachable points
  4. Mark remaining as noise

Best for: Arbitrary shapes, noise detection

Limitations: Parameter sensitivity, varying densities

Parameter Guidance

ε too small: Many small clusters, more noise

ε too large: Few large clusters, less noise

minPts too small: Noise as clusters

minPts too large: Core points missed
