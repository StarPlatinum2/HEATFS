# Unsupervised Learning Dashboard

A comprehensive web-based dashboard for exploring and visualizing unsupervised machine learning algorithms. This interactive tool allows you to experiment with K-Means clustering, Principal Component Analysis (PCA), and DBSCAN clustering using both synthetic and real datasets.
